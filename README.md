## 📋 Sobre

Portfólio profissional desenvolvido para apresentar minha trajetória como desenvolvedor Java Backend, projetos, habilidades técnicas e formas de contato.

Paleta de cores: **preto**, **verde** e **dourado** — com suporte a modo claro/escuro.

---

## ✨ Funcionalidades

- 🌙 Modo claro/escuro (toggle)
- ⌨️ Efeito typewriter no hero
- 🌐 Partículas animadas no fundo (canvas)
- 📜 Scroll reveal nas seções
- 📱 Layout responsivo (mobile-first)
- 🗂️ Timeline de carreira interativa
- 📬 Formulário de contato

## 🗂️ Estrutura do projeto

```
portfolio-rhaniel/
```
